# 🌟 Kids Learning App - Play & Learn English

## About This App
A fun and interactive learning app for kids aged 3-10 years. Children can learn English through games, speaking practice, stories, and writing activities.

## Features

### 🎮 Games
- Alphabet Hunter - Learn letters A-Z
- Word Builder - Build words from letters
- Match & Learn - Match words with pictures
- Sentence Maker - Arrange words to form sentences

### 🗣️ Speaking Practice
- Listen and repeat words with pronunciation
- Voice recognition to check speaking
- Daily conversation practice

### 📚 Stories
- Moral stories with new vocabulary words
- Audio pronunciation support
- Learn new words in context

### ✍️ Writing Practice
- Trace letters on canvas
- Practice writing words
- Drawing activities

## How to Use
1. Open `index.html` in any modern browser
2. Click on different tabs: Games, Speaking, Stories, Writing
3. Play games to earn points
4. Practice speaking using microphone
5. Read stories and learn new words
6. Practice writing letters

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Web Speech API for voice recognition
- Canvas API for drawing
